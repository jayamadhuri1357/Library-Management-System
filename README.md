 Library-Management-System
This project is a Library Management System developed using Java OOP concepts and Collections Framework. It allows users to add books, manage users, borrow and return books, and handle invalid operations using custom exception handling. The project also uses Java 8 features like Stream API and Lambda Expressions.
3) Features
  - Add Books
  - Add Users
  - Borrow Books
  - Return Books
  - Exception Handling
   - Stream API and Lambda Expressions

 4) Technologies Used

  - ArrayList
  - HashMap
  - Java 8 Features
