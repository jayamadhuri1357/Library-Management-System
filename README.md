1) Assignment 1 - Library Management System

2) Description
  This project is a simple Library Management System developed using Java OOP concepts and Collections Framework.

3) Features
  - Add Books
  - Add Users
  - Borrow Books
  - Return Books
  - Exception Handling
   - Stream API and Lambda Expressions

 4) Technologies Used
  - Java
  - ArrayList
  - HashMap
  - Java 8 Features
